สรุปโปรเจกต์
- Copilot Translator คือ Chrome Extension (frontend) ที่ใช้ React + TypeScript + Vite เป็น UI และมี backend ท้องถิ่นในโฟลเดอร์ [server/](server/) (Node.js + Express) เพื่อสื่อสารกับ GitHub Copilot SDK (@github/copilot-sdk). Extension ฉีด content script เข้าเว็บเพจเพื่ออ่านการเลือกข้อความและแสดง tooltip ที่ส่งคำขอไปยัง backend.

คำสั่งที่ใช้บ่อย
- ติดตั้ง dependencies (root/front-end)
  - npm install
- รัน frontend ในโหมดพัฒนา (Vite)
  - npm run dev
- สร้าง production build ของ extension
  - npm run build
  - หลัง build ให้ก็อปไฟล์ manifest และไฟล์ใน public ไปยัง dist ถ้า build script ไม่ทำให้แล้ว:
    - cp manifest.json dist/ && cp -r public/* dist/ 2>/dev/null || :
- ดูตัวอย่าง build (preview)
  - npm run preview
- ตรวจสอบ linting
  - npm run lint

Backend (server)
- ติดตั้งและรัน
  - cd server
  - npm install
  - npm run dev    # ใช้ nodemon ในการพัฒนา
  - npm start      # รันด้วย node index.js ใน production-like
- ตัวแปรสภาพแวดล้อมที่สำคัญ (ดู README): GH_TOKEN (ถ้าจำเป็น) และ PORT (ปกติ 5555)

การติดตั้ง extension ใน Chrome (สรุปจาก README)
1. สร้าง build (root): npm run build
2. คัดลอก manifest และ public ไปยัง dist (ถ้า needed)
3. เปิด chrome://extensions/ → เปิด Developer mode → Load unpacked → เลือกโฟลเดอร์ dist/
